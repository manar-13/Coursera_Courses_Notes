- 75%+ of data science job listings include Python
- Used by: IBM, Google, NASA, Facebook, Amazon, Spotify, Reddit, and more

- Clear and readable syntax — easy for beginners

## Introduction to R Language

### Open Source vs Free Software

| | Open Source | Free Software |
|--|-------------|---------------|
| **Championed by** | Open Source Initiative (OSI) | Free Software Foundation (FSF) |
| **Focus** | Business focused | Values focused |
| **Both** | Free to use, support collaboration, use similar licenses (e.g. GNU) |

> Python is **open source**. R is **free software**.

### Who Uses R
- Statisticians, mathematicians, and data miners
- Mostly popular in **academia**
- Companies: IBM, Google, Facebook, Microsoft, Bank of America, Ford, Uber

### Why Learn R
- Free to use — private, commercial, and public collaboration allowed
- Array-oriented syntax — easier to translate math into code
- World's largest repository of statistical knowledge
- 15,000+ publicly released packages for complex data analysis
- Integrates with: C++, Java, C, .Net, Python
- Stronger object-oriented programming than most statistical languages
- Most programmers learn R after a few years into their data science career

### R Communities
- useR, WhyR, SatRdays, R-Ladies
- R project website for conferences and events

## Introduction to SQL

### What is SQL
- Stands for **Structured Query Language** (pronounced "ess cue el" or "sequel")
- First appeared in **1974**, developed at **IBM** — older than Python and R by ~20 years
- A **non-procedural language** — scope is limited to querying and managing data
- Not a traditional programming language, but data scientists use it constantly

### What SQL Works With
- Designed for **relational databases** — data stored in 2D tables (rows and columns)
- Also works with NoSQL and big data repositories (due to its popularity)
- Behaves like an **interpreter between you and the database**

### SQL Language Elements
Clauses, Expressions, Predicates, Queries, Statements

### Why Learn SQL
- Required for roles like: Business Analyst, Data Analyst, Data Engineer
- Data is accessed **directly** — no need to copy it, which speeds up workflows
- **ANSI standard** — learn it once, apply it to many different databases
- Simple and powerful

### Popular SQL Databases
MySQL, IBM DB2, PostgreSQL, SQLite, Oracle, MariaDB, Microsoft SQL Server, Apache Open Office Base

> The syntax may change slightly depending on which database system you use.
> Focus on one database first, then expand from there.

## Other Languages for Data Science

### Java
- General-purpose, object-oriented, fast and scalable
- Runs on the **Java Virtual Machine (JVM)**
- Popular in enterprise environments

| Tool | Purpose |
|------|---------|
| Weka | Data mining |
| Java-ML | Machine learning |
| Apache MLlib | Scalable machine learning |
| Deeplearning4j | Deep learning |
| Hadoop | Big data processing and storage |

### Scala
- Built to fix Java's shortcomings — also runs on JVM
- Name = **Scala**ble + **La**nguage
- Supports functional programming

| Tool | Purpose |
|------|---------|
| Apache Spark | Fast cluster computing — designed to be faster than Hadoop |
| Shark | Query engine (inside Spark) |
| MLlib | Machine learning (inside Spark) |
| GraphX | Graph processing (inside Spark) |
| Spark Streaming | Real-time data streaming |

### C++
- Extension of C — improves processing speed and system control
- Used when real-time data delivery to customers is needed

| Tool | Purpose |
|------|---------|
| TensorFlow | Deep learning (built with C++, runs via Python) |
| MongoDB | NoSQL database for big data |
| Caffe | Deep learning algorithm repository |

### JavaScript
- General-purpose web language — NOT related to Java
- Extended beyond browsers via Node.js

| Tool | Purpose |
|------|---------|
| TensorFlow.js | ML and deep learning in browser and Node.js |
| brain.js / machinelearn.js | Built on top of TensorFlow.js |
| R-js | Rewrites R's linear algebra into TypeScript |

### Julia
- Designed at **MIT** for high-performance numerical analysis
- Speed of C or Fortran, ease of Python or R
- Compiled language — runs directly on the processor
- Can call libraries from: C, Go, Java, MATLAB, R, Fortran, Python
- Created in **2012** — still young but promising

| Tool | Purpose |
|------|---------|
| JuliaDB | Working with large persistent datasets |
