## What is Computer Programming?

- **Computer programming** — the process of giving instructions to a computer to perform an action or set of actions
- Programming languages are the words and symbols used to write those instructions
- Computers operate using **binary** (ones and zeros) — representing the on/off states of transistors
- A **transistor** is the most fundamental component of a computer — it controls the flow of electricity through a circuit

### Low-Level vs High-Level Languages

| Type | Description |
|------|-------------|
| **Low-level languages** | Difficult to use, require lots of training, often only work on specific machines |
| **High-level languages** | Simpler rules and structures (syntax), much more familiar to humans |

## What is Python?

- Python is a **high-level, general-purpose programming language**
- Created by Guido van Rossum — named after **Monty Python** (British comedy), not the snake
- Key qualities: **versatile, easy to learn, powerful, open source**
- Used by: data professionals, scientists, web developers

### Why Python is Popular
- Easy to learn and use
- Open source — free for anyone to use
- Has thousands of **libraries** that extend its capabilities
- Large and active community of users

### What is a Library?
- A **library** is a **reusable collection of code**
- Instead of writing code from scratch, you use a library that already has the function you need
- Examples range from simple (summing numbers) to complex (building neural networks for AI)

## Spreadsheets vs SQL vs Python — Comparison

| Key Question | Spreadsheets | SQL | Python |
|-------------|-------------|-----|--------|
| **What is it?** | Program using rows/columns with formulas | Database language to communicate with databases | General purpose language for data analysis, ML, web dev |
| **Primary advantage** | Variety of built-in visualization tools | Manipulate and reorganize data as needed | Extensive libraries (Pandas, NumPy) for data work |
| **Best for** | Smaller datasets | Larger datasets | Larger datasets |
| **Data source** | Entered manually or imported | External database | CSV, spreadsheets, databases, APIs |
| **Data stored** | Spreadsheet file on your computer | Inside tables in the database | Python file or Jupyter Notebook |
| **Formulas/functions** | Yes | Yes | Yes |
| **Visualizations** | Yes | Yes (needs extra tool like BI) | Yes (Seaborn, Matplotlib, Plotly) |

> All three tools are often used **together**. Example: use SQL to pull data → clean it in Python → export as spreadsheet for quick insights.

### What All Three Have in Common
- They all use **filters** to filter datasets
- They all use **functions**
